## tech stack
- TypeScript 4.1.3

## how to install

### clone this repository

```
# via http
$ git clone https://github.com/on3dd/adventofcode2020.git
```

or

```
# via ssh
$ git clone git@github.com:on3dd/adventofcode2020.git
```

### enter the project directory 

```
$ cd adventofcode2020
```

### install dependencies
```shell
$ npm install
```

## how to run solutions
```
# run ts-node with src of desired day, in example:
$ ts-node ./src/day-1/part_1.ts
```
